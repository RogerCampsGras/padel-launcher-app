# Aplicació mòbil per al control remot d’una màquina llançadora de pilotes de pàdel

Aquest repositori conté el codi font de l’aplicació mòbil desenvolupada per controlar remotament el prototip d’una màquina llançadora de pilotes de pàdel.

L’aplicació permet enviar consignes de velocitat, efecte i orientació al sistema de control del prototip mitjançant comunicació Bluetooth.

## Funcions principals

- Connexió amb el sistema de control.
- Enviament de consignes de velocitat.
- Selecció del tipus d’efecte.
- Control de l’orientació vertical i horitzontal.
- Activació i aturada del sistema.
